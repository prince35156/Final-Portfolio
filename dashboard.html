Data Visualization Dashboard

An interactive data visualization dashboard built in Tableau to transform raw CSV data into meaningful, easy-to-understand visual insights with filters and drill-down capabilities.

Project Goal

To take a complex dataset and make it understandable to any audience by building a clean, interactive dashboard that highlights key metrics and trends at a glance.

Technologies Used

  • Tableau Public (dashboard creation)
  • Microsoft Excel / CSV (data preparation)
  • Python (data cleaning with pandas)

Key Features

  • Bar charts, line graphs, and pie charts
  • Interactive filters for dynamic data exploration
  • Color-coded KPI summary cards
  • Trend analysis over time periods
  • Exportable as PDF or image

What I Learned

This project taught me data storytelling — how to choose the right chart for the right data, how to design for clarity, and how to clean and structure raw data before visualization.
