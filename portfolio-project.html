🌐 Portfolio Website

A multi-page personal portfolio website built entirely with HTML and CSS — no JavaScript. Features a modern dark theme, responsive layout, and CSS-only animations and hover effects.

Project Goal

To create a professional online presence that showcases my projects, skills, and certificates, while keeping the code beginner-friendly and easy to maintain.

Technologies Used

  • HTML5 — structure and content
  • CSS3 — styling, Flexbox layout, animations
  • Google Fonts — Syne & DM Sans typefaces
  • CSS Variables — for consistent theming

Key Features

  • 5 main pages with consistent navigation
  • Dark theme with purple and teal accent colors
  • CSS-only hover effects and transitions
  • Fully responsive on mobile and desktop
  • Clean, beginner-friendly, well-commented code

What I Learned

This project deepened my understanding of CSS Flexbox, CSS variables, responsive design, and how to organize a multi-page static website with consistent styling across all pages.
