🐍 Python Mini Projects

A growing collection of small Python programs, each designed to practice a specific concept — from basic logic to file handling, web scraping, and simple games.

Project Goal

To strengthen Python fundamentals by building real programs instead of just reading theory. Each mini project solves a small real-world problem.

Mini Projects Included

  • Calculator — Basic arithmetic with error handling
  • File Organizer — Sorts files into folders by extension
  • Web Scraper — Fetches headlines using BeautifulSoup
  • Quiz Game — Multiple-choice quiz with score tracking
  • Password Generator — Random strong password creator

Technologies Used

  • Python 3
  • Built-in libraries: os, random, string
  • Third-party: requests, BeautifulSoup4

What I Learned

These projects helped me get comfortable with Python's syntax, file I/O, working with libraries, exception handling, and writing clean and reusable functions.
