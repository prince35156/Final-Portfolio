🕶️ AR Project

This Augmented Reality project overlays interactive 3D digital content onto the physical world using marker-based AR detection. The goal was to explore the potential of mixed-reality experiences in education and product visualization.

Project Goal

To build a working AR demo that can recognize image markers and display relevant 3D models or information on top of them — accessible via any modern smartphone camera.

Technologies Used

  • HTML & CSS for the base web interface
  • AR.js (marker-based Augmented Reality)
  • A-Frame (3D web framework)
  • Blender (for creating 3D models)

Key Features

  • Recognizes custom image markers in real-time
  • Displays 3D models over detected markers
  • Runs entirely in the browser — no app install needed
  • Works on both Android and iOS devices

What I Learned

Working on this project gave me a solid understanding of how augmented reality works in the browser, how to use A-Frame for 3D scenes, and how marker tracking algorithms detect and position virtual objects.
