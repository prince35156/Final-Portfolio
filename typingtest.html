Typing Test — Bash

A fully terminal-based typing speed test written in Bash shell script, designed to run on any Linux/Unix system. No external tools or internet connection required.

Project Goal

To explore Bash scripting capabilities by building a practical and fun utility tool that measures typing speed (WPM) and accuracy directly in the terminal.

Technologies Used

  • Bash Shell Scripting
  • Linux terminal (tested on Ubuntu)
  • Built-in Bash utilities: date, read, echo
  • ANSI color codes for terminal UI

Key Features

  • Randomly selects a sentence from a word bank
  • Measures time taken using date +%s%N
  • Calculates WPM and accuracy percentage
  • Color-coded output for correct/incorrect characters
  • Leaderboard stored in a text file

What I Learned

This project gave me hands-on experience with Bash scripting, string manipulation, arithmetic in shell, user input handling, and creating simple text-based UIs in the terminal.
